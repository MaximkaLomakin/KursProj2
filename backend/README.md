# Backend сервер для приложения Погода

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env` на основе `.env.example`:

```
PORT=5000
JWT_SECRET=секретный_ключ_для_разработки
OPENWEATHER_API_KEY=ваш_ключ_от_openweathermap
```

## Запуск

```bash
npm start
```

Для разработки с автоперезагрузкой:
```bash
npm run dev
```

## API Endpoints

### Авторизация
- `POST /api/регистрация` - регистрация нового пользователя
- `POST /api/авторизация` - вход в систему

### Погода
- `GET /api/погода?город=Москва` - получение данных о погоде (требует авторизации)

### Избранное
- `GET /api/избранное` - получение списка избранных городов (требует авторизации)
- `POST /api/избранное` - добавление города в избранное (требует авторизации)
- `DELETE /api/избранное` - удаление города из избранного (требует авторизации)

### Админ панель
- `GET /api/админ/пользователи` - список всех пользователей (требует роль админа)
- `PUT /api/админ/пользователи/:id` - редактирование пользователя (требует роль админа)
- `GET /api/админ/популярные-запросы` - популярные запросы погоды (требует роль админа)
